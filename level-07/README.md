# Adding Pseudo-classes

Source URL: https://studio.code.org/courses/web-development-2025/units/1/lessons/18/levels/7

Melissa is making this web page about video game development. Can you help her add all four pseudo-class states for her hyperlink?
## Do This
1. Add a rule set in the [CSS file](style.css) for each of the pseudo-class states to style the `<a>` element in order to achieve the following styles:
- "orchid" color link the same size as the paragraph text
- link turns to the "whitesmoke" color with a "rebeccapurple" color background and larger text when the mouse hovers over it
- link turns red if the user clicks on it
- link turns "black" color with the "orchid" color background _as the user clicks_ on it
**Don't forget to refresh the preview to see if everything is working as expected.**
