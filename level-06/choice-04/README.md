# Adding Hover Styling

Source URL: https://studio.code.org/courses/web-development-2025/units/1/lessons/18/levels/6/sublevel/4

Alana is creating this page about Animal Kingdom oddities. She has links styled as buttons that the user can click on to learn more about each strange animal but she would like each button's style to change when the user hovers over them in the hope that it will draw their attention and motivate them to click on the links.
Can you help her give each button an appealing, yet unique style?
## Do This
**In the [HTML file](index.html):**
1. Give each `<a>` tag a different class attribute.
**In the [CSS file](style.css):**
1. Add a rule set for each hyperlink using its class attribute in combination with the "hover" pseudo-class.
2. Give each hyperlink a unique style. Consider using any of the following suggestions:
- Adding different font colors
- Adding different background colors
- Adding different font styles
- Increasing the font size
- Changing the border styles
- Changing the border colors
- Changing the border radius
3. Make sure to test the links on each page to ensure they are working as expected.
**Don't forget to refresh the preview to see if everything is working as expected.**
