# Anchor Tag States - "hover" & "active"

Source URL: https://studio.code.org/courses/web-development-2025/units/1/lessons/18/levels/3

We applied two of the four states for the anchor (`<a>`) tag in the previous level. Let's try the other two states - "hover" and "active."
- **hover**: when the mouse is hovered over, but not clicked, the link
- **active**: the moment the mouse is actively clicking down on the link
## Do This
**In the [CSS file](style.css):**
1. Add a rule to style the "hover" state of the `<a>` element.
- The selector of your rule should look like this: `a:hover`
- Add the `color` property with a dark color value of your choice.
- Add the `background-color` property with a bright color value of your choice.
- Add the `font-size` property with the value of your choice that is larger than 16px.
- **Optional:** Add the `border-radius` property to give the background color rounded edges.
- This will define what your link looks like when the mouse is hovering over it - Add any other properties of your choice.
<br>
2. Add a rule to style the "active" state of the `<a>` element.
- The selector of your rule should look like this: `a:active`
- Add the `color` property with a different color than the "hover" state.
- Add the `background-color` property with a different color than the "hover" state.
- Add the `text-decoration-style` property with a value of your choice.
- This will define what your link looks _as it is being clicked_ - Add any other properties of your choice.
**in the preview:**
1. Open [index.html](index.html) and start the live preview.
2. Refresh the preview to make sure your styles have been applied.
3. Hover over the link to see your "hover" styling.
4. **_Slowly_** click on the link to see your "active" styling.
