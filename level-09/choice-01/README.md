# Simple Hidden Tip with Hover

Source URL: https://studio.code.org/courses/web-development-2025/units/1/lessons/18/levels/9/sublevel/1

Not only can we use the `hover` pseudo-class to draw a user's attention to a hyperlink on our page, but we can also it to **style other elements too**!
Let's see how we can code our CSS to have a "tip" appear when we hover over a `<div>` element.
## Do This
**In the [CSS file](style.css):**
1. Add the following properties to the paragraph rule set:
- `display: none;`
- `background-color: yellow;`
2. Add a rule set to style the paragraph child of the div tag with the "hover" pseudo-class.
- Your selector should look like this: `div:hover p`
- Add the following property: `display: block;`
3. Hover over the `div` to see what happens!
**Don't forget to refresh the preview to see if everything is working as expected.**
