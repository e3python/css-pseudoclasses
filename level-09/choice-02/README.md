# Image Emphasis with Hover

Source URL: https://studio.code.org/courses/web-development-2025/units/1/lessons/18/levels/9/sublevel/2

<img src="https://images.code.org/e19b6fd29ec3cad6dd2a99669ff8a145-image-1707950878809.gif" style="width:300px; float:right;">
Not only can we use the `hover` pseudo-class to draw a user's attention to a hyperlink on our page, but we can also it to **style other elements too**!
Let's see how we can code our images with the "hover" pseudo-class to bring attention to them.
## Do This
**In the [CSS file](style.css):**
1. Add a rule set to style the images with the "hover" pseudo-class.
- Add the following properties:
- Increased width of the image
- "fushia" border color
- A defined `border-width` to at least 10px
- Add any other properties you think will make the image stand out and bring a user's attention to it!
3. Hover over the the images to see what happens!
**Don't forget to refresh the preview to see if everything is working as expected.**
