# Reveal The Cracked Message

Source URL: https://studio.code.org/courses/web-development-2025/units/1/lessons/18/levels/9/sublevel/3

Not only can we use the `hover` pseudo-class to draw a user's attention to a hyperlink on our page, but we can also it to **style other elements too**!
Let's see how we can style a `<div>` element to change the font and background color to make text appear!
## Do This
**In the [CSS file](style.css):**
1. Add a rule set to style the `div`with the "hover" pseudo-class.
- Add the following properties to the rule set:
- Set the color of the tex to "black"
- Set the background color to "yellow"
3. Hover over the `div` to see what happens!
**Don't forget to refresh the preview to see if everything is working as expected.**
